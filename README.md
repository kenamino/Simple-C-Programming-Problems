| 章节 | 题目范围 | 难度 | 内容 |
|:---|:---|:---|:---|
| **基础程序设计** | 1-5 | ★☆☆☆☆ | 四则运算、奇偶判断、最大值、累加、九九乘法表 |

## 使用方法

每道题目的详细描述见 [`problems.md`](./problems.md)，参考答案见 [`answer.md`](./answer.md)。

### 编译运行示例

```bash
# 编译
gcc answer/01_basic.c -o basic

# 运行
./basic
```

## 学习建议

1. **循序渐进** — 从第一章基础题开始，逐步提升难度
2. **独立思考** — 先尝试自己实现，再参考答案
3. **理解原理** — 不仅要会写，还要理解时间/空间复杂度
4. **动手实践** — 每道题都要实际编译运行
